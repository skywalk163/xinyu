# 心语 Playground

**心语语言的在线演示环境**

## 📖 快速启动

**首次使用？请查看 [快速启动指南](QUICKSTART.md)**

Windows用户可以直接双击 `start.bat` 启动！

## 🎯 功能特性

- ✅ **在线编写代码** - 在浏览器中直接编写心语代码
- ✅ **实时执行** - 点击运行按钮即可执行代码
- ✅ **示例代码** - 提供丰富的示例代码供学习
- ✅ **语法文档** - 内置完整的语法参考文档
- ✅ **错误提示** - 友好的错误信息提示
- ✅ **快捷键支持** - Ctrl+Enter 快速运行

## 🚀 快速开始

### 1. 安装依赖

**方法1：安装所有项目依赖（推荐）**

```bash
# 在项目根目录
pip install -r requirements.txt
```

**方法2：只安装Playground依赖**

```bash
pip install flask flask-cors
```

### 2. 启动服务器

**方法1：使用启动脚本（推荐）**

```bash
cd playground
python start.py
```

**方法2：直接运行服务器**

```bash
python playground/server.py
```

**方法3：从项目根目录运行**

```bash
python -m playground.start
```

### 3. 访问Playground

打开浏览器访问：http://localhost:5000

### 4. 停止服务器

在终端按 `Ctrl+C` 停止服务器

## 🎨 特色功能

### 1. 元数驱动解析

心语采用创新的元数驱动解析技术，支持无括号函数调用：

```yan
定义 平方 = 函 x：
  返回 x 相乘 x。
。

打印 平方 5。  # 无需括号！
```

### 2. 中文操作符

使用中文动词作为操作符，更符合中文思维：

```yan
3 相加 5    # 加法
10 相减 3   # 减法
4 相乘 5    # 乘法
20 相除 4   # 除法
```

### 3. 自然语法

语法设计贴近自然语言：

```yan
如果 成绩 大于等于 90 那么：
  打印 "优秀"。
否则：
  打印 "继续努力"。
。
```

## 📝 开发说明

### 目录结构

```
playground/
├── index.html      # 主页面
├── server.py       # 后端服务
├── start.py        # 启动脚本
└── README.md       # 说明文档
```

### 自定义配置

修改 `server.py` 可以：

- 更改端口号
- 添加更多API接口
- 自定义执行环境

### 添加新示例

在 `index.html` 的 `examples` 对象中添加：

```javascript
examples.new_example = `# 新示例标题
定义 x = 10。
打印 x。
`;
```

然后在侧边栏添加对应的HTML元素。
