```心语

```

- ✅ 添加递归函数定义

```心语
  。
```
```心语
。
```

```心语

    跳出。  # 没有跳出语句
  。
。
```

## 对比newlisp实现

### newlisp的汉诺塔
```言语言
定 汉诺塔 = 函 n 源 目标 辅助：
  若 等 n 1：
    印 连 "移动盘子从 " 源 " 到 " 目标。
  。
  若 大 n 1：
    汉诺塔 (减 n 1) 源 辅助 目标。
    印 连 "移动盘子从 " 源 " 到 " 目标。
    汉诺塔 (减 n 1) 辅助 目标 源。
  。
。

汉诺塔 3 "A" "C" "B"。
```

```心语
定义 汉诺塔 = 函 n：

    返回 1。

    返回 前一步 相乘 2 相加 1。
  。
。
```
