**核心逻辑**：
```python
def _collect_args_by_arity(self, arity: Arity) -> List[ASTNode]:
    args = []
    while not self._is_at_end():
        # 检查是否应该停止收集
        if self._should_stop_collecting_args():
            break
        # 检查元数是否已满足
        if arity.should_stop_collecting(len(args)):
            break
        # 解析参数
        arg = self._parse_primary()
        args.append(arg)
    return args
```

## 🏆 技术亮点

### 1. 元数驱动参数收集

**设计**：
- 根据元数自动收集参数
- 遇到操作符动词自动停止
- 参数数量自动验证

**优点**：
- 无歧义解析
- 符合中文习惯
- 灵活支持可变参数

### 2. 操作符动词识别

**实现**：
```python
if self._is_operator_verb(name):
    # 操作符动词在中缀位置，不应该作为函数调用
    self.pos -= 1
    return IdentifierNode(...)
```

**效果**：
- 正确区分操作符动词和函数动词
- 避免歧义
- 支持中缀表达式

### 3. 灵活的停止条件

**多层判断**：
1. 遇到操作符动词停止
2. 遇到终止符停止
3. 遇到操作符token停止
4. 元数满足后停止

**结果**：准确判断参数边界
