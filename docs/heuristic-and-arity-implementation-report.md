### 1. 启发式规则实现

**文件**：src/parser/parser.py

**实现内容**：
- 遇到操作符停止参数解析
- 遇到关键字停止参数解析
- 智能回溯机制
- 表达式参数支持

**启发式规则**：

**规则1：操作符停止**
```python
# 遇到这些操作符时停止参数解析
operator_tokens = (TokenType.PLUS, TokenType.MINUS, TokenType.MULTIPLY,
                  TokenType.DIVIDE, TokenType.MODULO,
                  TokenType.EQUALS, TokenType.NOT_EQUALS,
                  TokenType.LESS, TokenType.GREATER, TokenType.LESS_EQ,
                  TokenType.GREATER_EQ, 
                  TokenType.AND, TokenType.OR, TokenType.NOT,
                  TokenType.ASSIGN)
```

**规则2：关键字停止**
```python
# 遇到这些关键字时停止参数解析
keyword_tokens = (TokenType.VAR, TokenType.FUNCTION, TokenType.IF,
                 TokenType.FOR, TokenType.WHILE, TokenType.REPEAT,
                 TokenType.RETURN, TokenType.BREAK, TokenType.CONTINUE)
```

**规则3：智能回溯**
```python
# 如果解析出的表达式是二元操作，且左操作数是函数调用
# 这可能意味着我们解析过头了，需要回溯
if isinstance(arg, BinaryOpNode) and isinstance(arg.left, FunctionCallNode):
    # 回溯，只解析第一个参数
    self.pos = pos
    arg = self._parse_primary()
```

**效果**：
- 正确解析函数调用参数
- 避免参数解析错误
- 支持复杂表达式参数

### 2. 元数驱动解析系统

**文件**：src/parser/function_arity.py

**实现内容**：
- 函数元数表（FunctionArity）
- 元数类型（固定、可变、未知）
- 元数查询接口
- 函数注册接口

**元数表内容**：

**内置函数**：
- 打印：可变参数
- 输入：0参数
- 整数、浮点、字符串：1参数
- 最大值、最小值：可变参数
- 长度、排序、反转：1参数
- 范围：1-3参数

**高阶函数**：
- 皆（map）：2参数
- 只（filter）：2参数
- 归（reduce）：3参数

**操作符**：
- 相加、相减、相乘、相除：2参数
- 等于、大于、小于：2参数
- 并且、或者：2参数
- 非也：1参数

**元数类型**：
```python
class ArityType(Enum):
    FIXED = "fixed"        # 固定元数
    VARIABLE = "variable"  # 可变元数
    UNKNOWN = "unknown"    # 未知元数
```

**查询接口**：
```python
get_arity(func_name) -> FunctionArity
is_fixed_arity(func_name) -> bool
get_expected_args(func_name) -> int
register_function(name, arity_type, min_args, max_args)
```

## 学习newlisp/yan

### 核心思想

**元数驱动解析**：
- 每个动词声明参数数量（元数）
- 解析器根据元数自动收集参数
- 无需空格分词，支持连续书写

**示例**：
```yan
加1 2。    # 加的元数是2，自动收集1和2
乘3 4。    # 乘的元数是2，自动收集3和4
```
