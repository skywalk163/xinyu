### 2. 编译流程验证

**三阶段编译**：
1. 词法分析：代码 → Token序列 ✅
2. 语法分析：Token序列 → AST ✅
3. 代码生成：AST → Python代码 ✅

**示例**：
```yan
定义 x = 5。
```

↓ 词法分析

```
[
  {类型: "VAR", 值: "定义"},
  {类型: "IDENTIFIER", 值: "x"},
  {类型: "ASSIGN", 值: "="},
  {类型: "NUMBER", 值: "5"},
  {类型: "PERIOD", 值: "。"},
]
```

↓ 语法分析

```
{
  类型: "VarDef",
  数据: {
    名字: "x",
    初始值: {类型: "Number", 数据: {值: "5"}},
  },
}
```

↓ 代码生成

```python
x = 5
```
