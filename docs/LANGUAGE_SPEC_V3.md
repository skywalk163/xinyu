## 1. 元数驱动解析系统

### 1.1 概述

心语语言采用**元数驱动解析**（Arity-Driven Parsing）技术，解决了无括号函数调用的歧义问题。

**核心思想**：
- 每个动词（函数/操作符）声明参数数量（元数）
- 解析器根据元数自动收集参数
- 收集到足够参数后自动停止

### 1.2 元数类型

心语支持四种元数类型：

#### 固定元数（Fixed Arity）
必须有固定数量的参数。

```yan
平方根 16        # 1个参数
相加 3 5         # 2个参数
```

#### 可变元数（Variable Arity）
可以有任意数量的参数（最少min个）。

```yan
打印 "你好" "世界" "！"    # 3个参数
求和 1 2 3 4 5             # 5个参数
```

#### 最小元数（Minimum Arity）
最少需要min个参数，可以更多。

```yan
最大值 1 2 3     # 至少1个参数
```

#### 范围元数（Range Arity）
参数数量必须在[min, max]范围内。

```yan
范围 10          # 1个参数
范围 1 10        # 2个参数
范围 1 10 2      # 3个参数（步长）
```

## 5. 用户定义函数

### 5.1 函数定义

用户定义函数会自动推断元数并注册到动词注册表。

**语法**：
```yan
定义 函数名 = 函 参数1 参数2 ...：
  函数体。
。
```

**示例**：
```yan
# 单参数函数
定义 平方 = 函 x：
  返回 x 相乘 x。
。

# 双参数函数
定义 相加两数 = 函 a b：
  返回 a 相加 b。
。

# 无参数函数
定义 常量 = 函：
  返回 42。
。
```

### 5.2 函数调用

函数调用采用无括号语法，根据元数自动收集参数。

**示例**：
```yan
平方 5。              # 调用单参数函数
相加两数 3 5。         # 调用双参数函数
常量。                 # 调用无参数函数
```

### 5.3 递归函数

**示例**：
```yan
定义 阶乘 = 函 n：
  如果 n 小于等于 1 那么：
    返回 1。
  否则：
    返回 n 相乘 阶乘 n 相减 1。
  。
。

阶乘 5。              # 结果：120
```

**注意**：递归函数需要注意递归深度限制。

## 6. 表达式解析

### 6.1 操作符优先级

操作符优先级从高到低：

1. 括号 `()`
2. 一元操作符 `非也`
3. 乘除 `相乘`、`相除`、`取余`
4. 加减 `相加`、`相减`
5. 比较 `等于`、`不等`、`大于`、`小于`、`大于等于`、`小于等于`
6. 逻辑 `并且`、`或者`

### 6.2 表达式示例

```yan
# 算术表达式
3 相加 5 相乘 2        # 结果：13（乘法优先）
(3 相加 5) 相乘 2      # 结果：16（括号改变优先级）

# 比较表达式
5 大于 3 并且 10 小于 20    # 结果：真值

# 复杂表达式
定义 结果 = (10 相加 5) 相乘 2 相除 3。
```

### 6.3 函数调用在表达式中

```yan
# 函数调用作为操作数
平方根 16 相加 平方根 25    # 结果：9

# 函数调用作为参数
平方根 相加两数 3 5。       # 平方根(8) = 2.83
```

## 8. 元数驱动解析的优势

### 8.1 无歧义解析

元数驱动解析确保每个表达式都有唯一的解析结果。

**示例**：
```yan
斐波那契 n 相减 1 相加 斐波那契 n 相减 2
```

解析为：`(斐波那契(n-1)) + (斐波那契(n-2))`

因为：
- `斐波那契` 是函数动词，元数为1
- `相减` 是操作符动词，会停止参数收集
- `相加` 是操作符动词，会停止参数收集

### 8.2 灵活的参数数量

支持固定、可变、最小、范围四种元数类型，适应不同场景。

### 8.3 符合中文习惯

无需括号，无需特殊分隔符，自然流畅。

### 8.4 易于扩展

添加新函数只需注册元数，解析器自动处理。

## 9. 已知限制

### 9.1 递归深度

递归函数受Python递归深度限制（默认1000）。

**解决方案**：
- 使用尾递归优化
- 增加递归深度限制
- 使用迭代替代递归

### 9.2 操作符动词歧义

操作符动词在中缀和前缀位置有不同含义。

**示例**：
```yan
相加 3 5        # 前缀：函数调用，结果8
3 相加 5        # 中缀：操作符，结果8
```

**建议**：统一使用中缀形式。

### 9.3 未注册函数

未注册的函数默认使用可变元数。

**建议**：所有用户定义函数都会自动注册，无需手动处理。

## 11. 示例程序

```yan
# 变量定义

定义 y = 20。

# 算术运算

定义 差 = x 相减 y。
定义 积 = x 相乘 y。
定义 商 = x 相除 y。

# 打印结果
打印 和 差 积 商。
```

### 11.2 函数示例

```yan
# 定义函数
定义 平方 = 函 x：
  返回 x 相乘 x。
。

定义 立方 = 函 x：
  返回 x 相乘 x 相乘 x。
。

  返回 a 相加 b。
。
```

### 11.3 条件示例

```yan
定义 成绩 = 85。

如果 成绩 大于等于 90 那么：
  打印 "优秀"。
可选 成绩 大于等于 80 那么：
  打印 "良好"。
可选 成绩 大于等于 70 那么：
  打印 "中等"。
可选 成绩 大于等于 60 那么：
  打印 "及格"。
否则：
  打印 "不及格"。
。
```

### 11.4 循环示例

```yan

循环 数字 遍历 数字列表：
  打印 数字。
。

# 条件循环
定义 计数 = 0。
当满足 计数 小于 5：
  打印 计数。
  定义 计数 = 计数 相加 1。
。

# 重复执行
重复 5 次数：
  打印 "你好"。
。
```
