# 心语编程语言用户手册

## 简介

心语（Xin Yu）是一个专为中文使用者设计的编程语言，使用中文关键字和语法，让编程更加直观易懂。本手册将指导您如何使用心语进行编程。

## 快速开始

### 安装

1. 克隆仓库：
```bash
git clone https://github.com/yourusername/xinyu.git
cd xinyu
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 验证安装：
```bash
python -c "from src.main import main; print('心语安装成功！')"
```

### 第一个程序

创建一个名为`hello.xinyu`的文件：
```
印"你好，世界！"。
```

运行程序：
```bash
python src/main.py hello.xinyu
```

输出：
```
你好，世界！
```

## 控制流

### 条件语句

#### 如果-否则
```
变量 分数 = 85。

若 分数 >= 90 {
    印"优秀"。
} 否则若 分数 >= 80 {
    印"良好"。
} 否则若 分数 >= 60 {
    印"及格"。
} 否则 {
    印"不及格"。
}
```

#### 三元运算符
```
变量 年龄 = 18。
变量 状态 = 若 年龄 >= 18 则 "成人" 否则 "未成年"。
印状态。
```

### 循环语句

#### 当循环
```
变量 计数 = 0。

当 计数 < 5 {
    印"计数：" + 转字符串(计数)。
    计数 = 计数 + 1。
}
```

#### 对于循环
```
水果列表 = ["苹果", "香蕉", "橙子"]。

对于 水果 在 水果列表 {
    印"我喜欢吃" + 水果。
}
```

#### 范围循环
```
对于 i 在 范围(5) {
    印"第" + 转字符串(i) + "次"。
}
```

#### 中断和继续
```
对于 i 在 范围(10) {
    若 i == 3 {
        继续。  # 跳过3
    }
    
    若 i == 7 {
        中断。  # 跳出循环
    }
    
    印i。
}
```

## 函数

### 函数定义
```
函数 问候(姓名) {
    返回 "你好，" + 姓名 + "！"。

结果 = 问候("张三")。
印结果。  # 输出：你好，张三！
```

### 默认参数
```
函数 介绍(姓名, 年龄 = 20, 城市 = "北京") {
    返回 姓名 + "，" + 转字符串(年龄) + "岁，来自" + 城市。

印介绍("李四")。           # 李四，20岁，来自北京

印介绍("赵六", 30, "上海")。 # 赵六，30岁，来自上海
```

### 可变参数
```
函数 求和(...数字) {
    结果 = 0。
    对于 数 在 数字 {
        结果 = 结果 + 数。
    }
    返回 结果。
}

印求和(1, 2, 3)。      # 6
印求和(10, 20, 30, 40)。 # 100
```

### 匿名函数
```
平方 = 函数(x) { 返回 x * x }。
印平方(5)。  # 25

排序列表 = 排序([3, 1, 4, 1, 5], 键=函数(x) { 返回 x })。
印排序列表。  # [1, 1, 3, 4, 5]
```

## 数据结构

### 列表操作

#### 访问元素
```
列表 = [10, 20, 30, 40, 50]。
第一个 = 列表[0]。      # 10
最后一个 = 列表[-1]。    # 50
切片 = 列表[1:4]。      # [20, 30, 40]
```

#### 修改列表
```
列表 = []。
列表.追加(1)。          # [1]
列表.追加(2)。          # [1, 2]
列表.插入(1, 1.5)。     # [1, 1.5, 2]
删除的 = 列表.弹出()。   # 2, 列表变为[1, 1.5]
列表.移除(1)。          # 列表变为[1.5]
```

#### 列表推导式
```
平方列表 = [x * x 对于 x 在 范围(5)]。  # [0, 1, 4, 9, 16]
偶数列表 = [x 对于 x 在 范围(10) 若 x % 2 == 0]。  # [0, 2, 4, 6, 8]
```

### 字典操作

#### 访问和修改

#### 遍历字典
```

```

### 集合操作
```
集合1 = {1, 2, 3, 4, 5}。
集合2 = {4, 5, 6, 7, 8}。
```

## 错误处理

### 基本错误处理
```

    结果 = 10 / 0。
    印结果。
} 捕获 错误 {
    印"发生错误：" + 转字符串(错误)。
} 最终 {
    印"清理完成。"。
}
```

### 特定错误类型
```
尝试 {
    文件 = 打开("不存在的文件.txt", "r")。
    内容 = 文件.读取()。
    文件.关闭()。
} 捕获 文件未找到错误 为 e {
    印"文件不存在：" + e.文件名。
} 捕获 权限错误 为 e {
    印"没有权限：" + e.文件名。
} 捕获 错误 为 e {
    印"其他错误：" + 转字符串(e)。
}
```

### 抛出错误
```
函数 检查年龄(年龄) {
    若 年龄 < 0 {
        抛出 值错误("年龄不能为负数")。
    }
    若 年龄 > 150 {
        抛出 值错误("年龄不能超过150岁")。
    }
    返回 年龄。
}

尝试 {
    检查年龄(-5)。
} 捕获 值错误 为 e {
    印"无效年龄：" + e.消息。
```


## 最佳实践

### 代码组织

1. **模块化设计**
   - 将相关功能组织到模块中
   - 每个模块专注于单一职责
   - 使用有意义的模块名

2. **函数设计**
   - 函数应该小而专注
   - 使用有意义的参数名
   - 添加文档字符串

3. **错误处理**
   - 使用具体的错误类型
   - 提供有用的错误信息
   - 不要忽略错误

### 性能优化

1. **避免重复计算**
   ```
   # 不好
   对于 i 在 范围(长度(列表)) {
       印列表[i]。
   }
   
   # 好
   长度 = 长度(列表)。
   对于 i 在 范围(长度) {
       印列表[i]。
   }
   ```

2. **使用适当的数据结构**
   - 频繁查找使用字典
   - 有序数据使用列表
   - 去重使用集合

3. **惰性求值**
   - 使用生成器处理大数据
   - 延迟计算直到需要

### 代码风格

1. **命名约定**
   - 变量：小写字母，下划线分隔
   - 函数：小写字母，下划线分隔
   - 类：驼峰命名法
   - 常量：大写字母，下划线分隔

2. **注释规范**
   - 每个函数应该有文档字符串
   - 复杂逻辑需要注释
   - 避免无意义的注释

3. **代码格式化**
   - 使用4个空格缩进
   - 行长度不超过80字符
   - 操作符周围有空格

### 待办事项应用
```
类 待办事项 {
    构造() {
        自我.任务列表 = []。
    }
    
    函数 添加任务(描述) {
        任务 = {
            "描述": 描述,
            "完成": 假,
            "创建时间": 时间.时间()
        }。
        自我.任务列表.追加(任务)。
        印"已添加任务：" + 描述。
    }
    
    函数 完成任务(索引) {
        若 索引 < 0 或 索引 >= 长度(自我.任务列表) {
            印"无效的任务索引"。
            返回。
        }
        
        任务 = 自我.任务列表[索引]。
        任务["完成"] = 真。
        印"已完成任务：" + 任务["描述"]。
    }
    
    函数 显示任务() {
        印"\\n=== 待办事项 ==="。
        
        若 长度(自我.任务列表) == 0 {
            印"没有待办事项"。
            返回。
        }
        
        对于 i, 任务 在 枚举(自我.任务列表) {
            状态 = 若 任务["完成"] 则 "[✓]" 否则 "[ ]"。
            印转字符串(i) + ". " + 状态 + " " + 任务["描述"]。
        }
    }
    
    函数 删除任务(索引) {
        若 索引 < 0 或 索引 >= 长度(自我.任务列表) {
            印"无效的任务索引"。
            返回。
        }
        
        任务 = 自我.任务列表.弹出(索引)。
        印"已删除任务：" + 任务["描述"]。
    }
}

# 使用示例

待办事项.添加任务("编写示例代码")
待办事项.添加任务("测试程序")
```

## 故障排除

### 常见错误

1. **语法错误**
   ```
   # 错误：缺少句号
   变量 x = 10
   
   # 正确：
   变量 x = 10。
   ```

2. **名称错误**
   ```
   # 错误：变量未定义
   印y。
   
   # 正确：
   变量 y = 20。
   印y。
   ```

3. **类型错误**
   ```
   # 错误：字符串和数字不能相加
   结果 = "数字：" + 42。
   
   # 正确：
   结果 = "数字：" + 转字符串(42)。
   ```

### 调试技巧

1. **使用打印语句**
   ```
   变量 x = 10。
   印"x的值：" + 转字符串(x)。
   ```

2. **检查变量类型**
   ```
   变量 数据 = [1, 2, 3]。
   印"数据类型：" + 类型(数据)。
   ```

3. **使用断言**
   ```
   函数 计算平均值(数字列表) {
       断言 长度(数字列表) > 0, "数字列表不能为空"
       总和 = 0。
       对于 数 在 数字列表 {
           总和 = 总和 + 数。
       }
       返回 总和 / 长度(数字列表)。
   }
   ```

## 开发工具

### 代码格式化工具

心语提供了强大的代码格式化工具，帮助您保持代码风格一致。

#### 安装格式化工具
```bash
# 安装格式化工具
python tools/setup_formatter.py

# 或直接使用
python tools/xinyu_format.py --help
```

#### 基本使用
```bash
# 格式化单个文件
python tools/xinyu_format.py format example.xinyu

# 检查代码格式（不修改文件）
python tools/xinyu_format.py check example.xinyu

# 格式化目录下所有文件
python tools/xinyu_format.py format src/

# 格式化并显示差异
python tools/xinyu_format.py diff example.xinyu
```

#### 配置格式化规则
创建 `.xinyu-formatter.yaml` 文件来自定义格式化规则：
```yaml
# 缩进设置
indent_size: 4
use_tabs: false

# 空格设置
space_around_operators: true
space_after_comma: true
space_after_colon: true

# 换行设置
max_line_length: 100
wrap_long_lines: true

# 其他设置
remove_trailing_whitespace: true
ensure_newline_at_eof: true
```

```
心语> 历史 导入 <文件>          # 从文件导入历史记录
心语> 历史 编辑 <编号> <新命令> # 编辑历史记录并重新执行
心语> 历史 清除                 # 清除所有历史记录
```

#### 命令类型支持
- `expression`  - 表达式求值
- `statement`   - 语句执行
- `definition`  - 定义（函数、变量等）
- `import`      - 导入语句
- `control`     - 控制语句（if、for等）
- `debug`       - 调试命令
- `help`        - 帮助命令
- `other`       - 其他命令
