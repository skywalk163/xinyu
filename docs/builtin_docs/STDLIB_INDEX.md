## 使用方法

### 基本用法

```python
from src.module import ModuleManager

manager = ModuleManager()

# 使用中文名导入
数学 = manager.import_module('数学')
print(数学.平方根(16))  # 4.0

# 使用英文名导入（也可以）
math = manager.import_module('math')
print(math.sqrt(16))  # 4.0
```

### 查看所有可用模块

```python
manager = ModuleManager()
modules = manager._chinese_module_map
for chinese_name, english_name in modules.items():
    print(f"{chinese_name} -> {english_name}")
```

1. **中英文双语支持** - 所有模块都支持中文和英文名称

3. **完整封装** - 提供常用函数和类的中文接口
