## 兼容性设计

### 双语法支持
系统现在同时支持新旧两种语法：

**新语法（推荐）**：
```yan
定义 x = 5。
函数 平方：
  参数 x。
  返回 x 相乘 x。
打印 平方 5。
```

**旧语法（兼容）**：
```yan
定 x = 5。
函 平方：
  接收 x。
  返回 x 乘 x。
印 平方 5。
```

### 优势
1. **平滑迁移**：用户可以逐步迁移到新语法
2. **向后兼容**：旧代码仍然可以正常运行
3. **灵活性**：新旧语法可以混用
4. **易维护**：统一的映射表，易于管理
