**文档内容**：
- **架构概述**：多阶段编译器架构，前端（分析）+ 后端（生成）
- **模块划分**：8个核心模块，职责清晰
- **数据流**：源代码 → Token → AST → Python代码 → 执行结果
- **接口契约**：定义了ILexer、IParser、ISemanticAnalyzer等接口
- **依赖关系**：单向依赖，无循环
- **扩展点**：添加新关键字、操作符、AST节点、宏的指南

**模块职责边界**：
- **Lexer**：词法分析，不负责语法和语义
- **Parser**：语法分析，不负责词法和语义
- **SemanticAnalyzer**：语义分析，不负责语法和代码生成
- **PythonCodegen**：代码生成，不负责语义和执行
- **Runtime**：执行代码，不负责代码生成
- **Security**：输入验证，独立模块

## 使用指南

### 安装开发工具

```bash
pip install black isort mypy flake8 pre-commit bandit
```

### 安装pre-commit钩子

```bash
pre-commit install
```

### 运行代码格式化

```bash
# 格式化所有代码
black src/ tests/

# 排序导入
isort src/ tests/

# 类型检查
mypy src/
```

### 运行所有检查

```bash
# 运行pre-commit所有检查
pre-commit run --all-files
```

### 运行测试

```bash
# 运行所有测试
pytest

# 运行测试并生成覆盖率报告
pytest --cov=src --cov-report=html
```
