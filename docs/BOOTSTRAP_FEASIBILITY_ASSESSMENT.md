## 自举实现计划

### 1. 词法分析器（中文实现）

**文件**：selfhost/lexer.yan

**核心函数**：
```yan
函数 词法分析：
  参数 代码。
  定义 tokens = []。
  定义 位置 = 0。
  
  当满足 位置 小于 长度 代码：
    定义 字符 = 代码[位置]。
    
    # 识别关键字
    如果 是关键字 字符 那么
      定义 token = 识别关键字 代码 位置。
      tokens.添加 token。
      位置 = 位置 相加 token.长度。
    否则
      # 其他处理
      位置 = 位置 相加 1。
  
  返回 tokens。
```

**需要实现**：
- 识别关键字
- 识别操作符
- 识别标识符
- 识别数字
- 识别字符串
- 跳过空白

### 2. 语法分析器（中文实现）

**文件**：selfhost/parser.yan

**核心函数**：
```yan
函数 语法分析：
  参数 tokens。
  定义 位置 = 0。
  定义 ast = []。
  
  当满足 位置 小于 长度 tokens：
    定义 token = tokens[位置]。
    
    # 解析语句
    如果 token.类型 等于 "定义" 那么
      定义 节点 = 解析变量定义 tokens 位置。
      ast.添加 节点。
      位置 = 节点.结束位置。
    否则
      # 其他处理
      位置 = 位置 相加 1。
  
  返回 ast。
```

**需要实现**：
- 解析变量定义
- 解析函数定义
- 解析条件语句
- 解析循环语句
- 解析表达式

### 3. 代码生成器（中文实现）

**文件**：selfhost/codegen.yan

**核心函数**：
```yan
函数 代码生成：
  参数 ast。
  定义 代码 = ""。
  
  循环 节点 遍历 ast：
    如果 节点.类型 等于 "变量定义" 那么
      代码 = 代码 相加 生成变量定义 节点。
    否则
      # 其他处理
      代码 = 代码 相加 ""。
  
  返回 代码。
```

**需要实现**：
- 生成变量定义
