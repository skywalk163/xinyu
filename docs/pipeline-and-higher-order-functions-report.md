
**管道语法**（设计）：
```yan
# 管道操作示例
列表 1 2 3，皆 相乘 2，只 大于 3，归 相加 0。

# 等价于
归(只(皆(列表 1 2 3, 相乘 2), 大于 3), 相加 0)
```

**注入规则**：
- 固定元数函数：管道值作为第一个参数
- 可变参数函数：管道值注入到最后一个参数位

### 2. 高阶函数

**TokenType定义**：
```python

MAP = auto()         # 皆（映射）

REDUCE = auto()      # 归（归约）
```

**关键字映射**：
```python
# 高阶函数
"皆": TokenType.MAP,          # map
"只": TokenType.FILTER,       # filter
"归": TokenType.REDUCE,       # reduce
```

**高阶函数设计**：

**皆（map）**：
```yan
# 映射：对列表每个元素应用函数
列表 1 2 3，皆 相乘 2。  # [2, 4, 6]
```

**只（filter）**：
```yan
# 筛选：保留满足条件的元素
列表 1 2 3 4 5，只 大于 3。  # [4, 5]
```

**归（reduce）**：
```yan
# 归约：将列表归约为单个值
列表 1 2 3 4 5，归 相加 0。  # 15
```

**设计差异**：
- newlisp/yan：使用单字关键字（列、皆）
- 心语：使用双字关键字（列表、皆）

## 使用示例

### 管道操作示例

**基本管道**：
```yan
# 映射
列表 1 2 3，皆 相乘 2。  # [2, 4, 6]

# 筛选
列表 1 2 3 4 5，只 大于 3。  # [4, 5]

# 归约
列表 1 2 3 4 5，归 相加 0。  # 15
```

**链式管道**：
```yan
# 链式操作
列表 1 2 3 4 5，皆 相乘 2，只 大于 5，归 相加 0。
# [2, 4, 6, 8, 10] → [6, 8, 10] → 24
```

### 高阶函数示例

**映射（皆）**：
```yan
# 平方所有元素
列表 1 2 3 4 5，皆 平方。  # [1, 4, 9, 16, 25]

# 自定义函数
定义 双倍 = 函数 x：返回 x 相乘 2。
列表 1 2 3，皆 双倍。  # [2, 4, 6]
```

**筛选（只）**：
```yan
# 筛选偶数
列表 1 2 3 4 5 6，只 是偶数。  # [2, 4, 6]

# 筛选正数
列表 -3 -2 -1 0 1 2 3，只 大于 0。  # [1, 2, 3]
```

**归约（归）**：
```yan
# 求和
列表 1 2 3 4 5，归 相加 0。  # 15

# 求积
列表 1 2 3 4 5，归 相乘 1。  # 120

# 最大值
列表 3 1 4 1 5 9 2 6，归 最大值 0。  # 9
```
