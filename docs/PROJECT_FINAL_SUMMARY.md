## 技术亮点

### 1. 语法设计

**双字关键字**：
- 定义、函数、如果、真值、假值
- 语义明确，易于理解
- 新旧语法兼容

**启发式规则**：
- 遇到操作符停止参数解析
- 遇到关键字停止参数解析
- 智能回溯机制

**元数驱动解析**：
- 函数元数表（50+函数）
- 支持固定元数和可变元数
- 为无空格分词奠定基础

### 2. 多轨制系统

**四种轨道**：
- 数学轨：`$(...)` - 数学表达式
- Python轨：`{{...}}` - Python代码
- SQL轨：`SQL"..."` - SQL查询
- Shell轨：`>(...)` - Shell命令

**优势**：
- 灵活性：不同任务使用最适合的轨道
- 性能：数学表达式直接计算
- 兼容性：可直接使用Python生态

### 3. 标准库

**四个模块**：
- string.yan：字符串操作（30+函数）
- math.yan：数学函数（已有）
- io.yan：文件操作（40+函数）
- net.yan：网络操作（30+函数）

**覆盖场景**：
- 字符串处理
- 数学计算
- 文件IO
- 网络请求

### 4. 函数式编程

**管道操作**：
```yan
列表 1 2 3，皆 相乘 2，只 大于 3，归 相加 0。
```

**高阶函数**：
- 皆（map）：映射
- 只（filter）：筛选
- 归（reduce）：归约

### 5. 安全性

**多层防御**：
- 输入验证
- 编译时检查
- 运行时隔离
- RestrictedPython

### 6. 性能优化

**优化措施**：
- 优化词法分析器（2-5倍提升）
- 编译缓存（LRU策略）
- 垃圾回收（引用计数+标记清除）

## 使用指南

### 安装

```bash
git clone https://gitcode.com/skywalk163/xinyu.git
cd xinyu
pip install -r requirements.txt
```

### 快速开始

**Hello World**：
```yan
打印 "你好，世界！"。
```

**变量定义**：
```yan
定义 x = 5。
定义 名字 = "张三"。
打印 x。
打印 名字。
```

**函数定义**：
```yan
函数 平方：
  参数 x。
  返回 x 相乘 x。

打印 平方 5。
```

**条件语句**：
```yan
如果 x 大于 5 那么
  打印 "大于5"。
否则
  打印 "小于等于5"。
```

**循环**：
```yan
循环 i 遍历 [1, 2, 3]：
  打印 i。

当满足 x 大于 0：
  打印 x。
  x = x 相减 1。
```

### 使用REPL

```bash
python tools/repl.py

心语> 定义 x = 5。
心语> 打印 x。
5
心语> .help
心语> .exit
```

### 使用VS Code插件

```bash
cd tools/vscode-extension
npm install
npm run compile
# 在VS Code中按F5启动调试
```

### 使用包管理器

```bash
python tools/package_manager.py install http
python tools/package_manager.py list
python tools/package_manager.py search json
```
