### 成果展示

#### 元数系统使用示例
```python
from src.parser.arity import Arity

# 固定元数
arity = Arity.fixed(2)
assert arity.is_satisfied(2) == True
assert arity.should_stop_collecting(2) == True

# 可变元数
arity = Arity.variable(min=0)
assert arity.is_satisfied(5) == True
assert arity.should_stop_collecting(5) == False

# 范围元数
arity = Arity.range(min=1, max=3)
assert arity.is_satisfied(2) == True
assert arity.should_stop_collecting(3) == True
```

#### 动词注册表使用示例
```python
from src.parser.verb_registry import VerbRegistry
from src.parser.arity import Arity

registry = VerbRegistry()
registry.register_builtin_verbs()

# 查询元数
arity = registry.get("相加")  # Arity.fixed(2)

# 判断类型
registry.is_operator("相加")  # True
registry.is_function("打印")  # True
```
