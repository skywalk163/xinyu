### 2. 新增示例 ✅

#### 汉诺塔（Hanoi Tower）

```yan
# 汉诺塔问题（打印移动步骤）
打印 "汉诺塔3个盘子的移动步骤："。
打印 "第1步：移动盘子1 从A到C"。
打印 "第2步：移动盘子2 从A到B"。
...
打印 "汉诺塔递归公式："。
打印 "移动次数 = 2^n - 1"。
```

**教学价值**：
- 理解递归思想
- 学习问题分解
- 掌握算法复杂度

#### 冒泡排序（Bubble Sort）

```yan
# 冒泡排序
打印 "冒泡排序演示"。
打印 "排序原理："。
打印 "1. 比较相邻元素"。
打印 "2. 如果前者大于后者，交换位置"。
打印 "3. 重复直到数组有序"。
打印 "4. 时间复杂度：O(n的平方)"。
```

**教学价值**：
- 理解排序算法
- 学习比较和交换
- 掌握时间复杂度

#### 图灵机（Turing Machine）

```yan
# 图灵机模拟（二进制加1）
打印 "图灵机：二进制加1"。
打印 "输入纸带：1011 (二进制11)"。
打印 "执行步骤："。
打印 "1. 移到最右端"。
打印 "2. 当前位是1，写0，左移"。
...
打印 "输出纸带：1100 (二进制12)"。
```

**教学价值**：
- 理解计算理论
- 学习状态机模型
- 掌握二进制运算

#### 素数筛（Prime Sieve）

```yan
# 埃拉托斯特尼素数筛
打印 "埃拉托斯特尼素数筛"。
打印 "原理：筛除所有合数，剩下的就是素数"。
打印 "筛法过程："。
打印 "1. 从2开始，2是素数"。
打印 "2. 筛除2的倍数：4, 6, 8, 10, ..."。
...
打印 "2到30之间的素数："。
打印 2。打印 3。打印 5。...
```

**教学价值**：
- 理解数论基础
- 学习筛法思想
- 掌握算法优化
