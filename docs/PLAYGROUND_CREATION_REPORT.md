## 🔧 API接口

### 1. 执行代码

**请求**：
```
POST /api/execute
Content-Type: application/json

{
  "code": "打印 \"你好\"。"
}
```

**响应**：
```json
{
  "success": true,
  "output": ["你好"],
  "python_code": "print(\"你好\")"
}
```

### 2. 编译代码

**请求**：
```
POST /api/compile
Content-Type: application/json

{
  "code": "打印 \"你好\"。"
}
```

**响应**：
```json
{
  "success": true,
  "python_code": "print(\"你好\")"
}
```

## 📝 示例代码展示

### 示例1：你好，世界

```yan
# 你好，世界
定义 问候 = "你好，心语！"。
打印 问候。

定义 名字 = "世界"。
打印 "你好，" 名字 "！"。
```

### 示例2：函数定义

```yan
# 函数定义示例
定义 平方 = 函 x：
  返回 x 相乘 x。
。

定义 立方 = 函 x：
  返回 x 相乘 x 相乘 x。
。

打印 "平方(5) = " 平方 5。
打印 "立方(3) = " 立方 3。
```

### 示例3：斐波那契

```yan
# 斐波那契数列
定义 斐波那契 = 函 n：
  如果 n 小于 2 那么：
    返回 n。
  。
  返回 斐波那契 n 相减 1 相加 斐波那契 n 相减 2。
。

打印 "斐波那契数列前10项："。
遍历 i 于 范围 1 11：
  打印 "斐波那契(" i ") = " 斐波那契 i。
。
```
