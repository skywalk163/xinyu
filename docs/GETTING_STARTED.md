# 心语快速开始指南

## 安装

### 1. 克隆仓库

```bash
git clone https://gitcode.com/skywalk163/xinyu.git
cd xinyu
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 验证安装

```bash
python -m pytest tests/ -v
```

## 基础语法

### 1. 变量定义

```yan
# 新语法（推荐）
定义 x = 5。
定义 名字 = "张三"。
定义 价格 = 3.14。

# 旧语法（兼容）
定 x = 5。
定 名字 = "张三"。
```

### 2. 数据类型

```yan
# 数字
定义 整数 = 42。
定义 浮点数 = 3.14。

# 字符串
定义 文本 = "你好"。

# 布尔值
定义 是真 = 真值。
定义 是假 = 假值。

# 列表
定义 列表 = [1, 2, 3, 4, 5]。

# 字典
定义 字典 = {"名字": "张三", "年龄": 25}。
```

### 3. 运算

```yan
# 算术运算
定义 和 = 1 相加 2。      # 3
定义 差 = 5 相减 3。      # 2
定义 积 = 4 相乘 3。      # 12
定义 商 = 10 相除 2。     # 5.0

# 比较运算
定义 相等 = 5 等于 5。    # 真值
定义 更大 = 10 大于 5。   # 真值

# 逻辑运算
定义 结果 = 真值 并且 假值。  # 假值
定义 取反 = 非也 真值。      # 假值
```

### 4. 条件语句

```yan
定义 x = 10。

如果 x 大于 5 那么
  打印 "x 大于 5"。
否则
  打印 "x 不大于 5"。
```

### 5. 循环

**遍历循环**：

```yan
循环 i 遍历 [1, 2, 3]：
  打印 i。
```

**条件循环**：

```yan
定义 x = 5。
当满足 x 大于 0：
  打印 x。
  x = x 相减 1。
```

**重复执行**：

```yan
重复 3 次数：
  打印 "你好"。
```

## 常见问题

### Q: 新旧语法可以混用吗？

A: 可以。系统同时支持新旧两种语法，可以平滑迁移。

```yan
定义 x = 5。    # 新语法
定 y = 10。     # 旧语法
打印 x 相加 y。  # 正常工作
```

### Q: 如何处理错误？

A: 心语会提供详细的错误信息：

```
语法错误: Expected '那么' after condition (行 1, 列 6)
  💡 建议: 条件语句需要使用 '那么' 关键字
```

```bash
ls examples/

```
