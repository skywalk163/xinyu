## 简介

**心语**（Xīn Yǔ）是一门极简的中文编程语言，寓意"从心而发，自然表达"。它只有5个核心关键字，让代码如心声般自然流淌。

### 核心特性

- **极简关键字**：只有5个核心关键字（定、函、若、真、假）
- **中文操作符**：使用中文操作符，更符合中文思维
- **语境驱动**：支持语境省略，让代码更贴近自然语言
- **函数式编程**：支持高阶函数、递归、管道操作
- **Python互操作**：生成可执行的Python代码

## 基础语法

### 1. 变量定义

使用 `定` 关键字定义变量：

```
定 x = 10。
定 姓名 = "张三"。
定 分数 = 95.5。
```

语境驱动模式下，可以直接赋值：

```
年龄 = 25。
印 年龄。
```

### 2. 数据类型

心语支持以下数据类型：

- **数字**：整数和浮点数
  ```
  定 整数 = 123。
  定 浮点数 = 3.14。
  ```

- **字符串**：使用双引号
  ```
  定 文本 = "你好，心语"。
  ```

- **布尔值**：`真` 和 `假`
  ```
  定 开关 = 真。
  定 关闭 = 假。
  ```

- **列表**：使用方括号
  ```
  定 数字列表 = 【1， 2， 3， 4， 5】。
  定 字符串列表 = 【"苹果"， "香蕉"， "橙子"】。
  ```

- **字典**：使用花括号
  ```
  定 学生 = ｛"姓名"： "张三"， "年龄"： 20｝。
  ```

### 3. 运算符

#### 算术运算符

| 心语 | Python | 示例 |
|------|--------|------|
| 加 | + | `1加2` → 3 |
| 减 | - | `5减3` → 2 |
| 乘 | * | `4乘5` → 20 |
| 除以 | / | `10除以2` → 5 |
| 取模 | % | `10取模3` → 1 |

#### 比较运算符

| 心语 | Python | 示例 |
|------|--------|------|
| 等于 | == | `5等于5` → 真 |
| 不等于 | != | `5不等于3` → 真 |
| 大于 | > | `5大于3` → 真 |
| 小于 | < | `3小于5` → 真 |
| 大于等于 | >= | `5大于等于5` → 真 |
| 小于等于 | <= | `3小于等于5` → 真 |

#### 逻辑运算符

| 心语 | Python | 示例 |
|------|--------|------|
| 且 | and | `真且假` → 假 |
| 或 | or | `真或假` → 真 |
| 非 | not | `非真` → 假 |

### 4. 条件判断

使用 `若` 关键字：

```
# 简单条件
若 x大于0：
  印 "正数"。
。

# if-else
若 x大于0：
  印 "正数"。
否则：
  印 "非正数"。
。

# if-elif-else
若 分数大于等于90：
  印 "优秀"。
否则若 分数大于等于60：
  印 "及格"。
否则：
  印 "不及格"。
。
```

### 5. 循环

#### 遍历循环

```
遍历 i 于 【1， 2， 3， 4， 5】：
  印 i。
。

遍历 i 于 范围(1， 10)：
  印 i。
。
```

#### 当循环

```
定 计数 = 0。
当 计数小于5：
  印 计数。
  计数 = 计数加1。
。
```

#### 重复循环

```
重复 3 次：
  印 "你好！"。
。
```

### 6. 函数定义

使用 `函` 关键字：

```
# 无参数函数
定 问好 = 函：
  印 "你好！"。
。

# 单参数函数
定 平方 = 函 x：
  返回 x乘x。
。

# 多参数函数
定 相加 = 函 a， b：
  返回 a加b。
。

# 递归函数
定 阶乘 = 函 n：
  若 n小于等于1：
    返回 1。
  否则：
    返回 n乘阶乘(n减1)。
  。
。
```

## 进阶特性

### 1. 列表操作

```
定 列表 = 【1， 2， 3， 4， 5】。

# 访问元素
印 列表[0]。        # 第一个元素
印 列表[负1]。      # 最后一个元素

# 列表方法
列表.添加(6)。      # 添加元素
印 列表长度(列表)。  # 长度
印 求和(列表)。      # 求和
印 最大值(列表)。    # 最大值
印 最小值(列表)。    # 最小值
印 排序(列表)。      # 排序
```

### 2. 字典操作

```
定 学生 = ｛"姓名"： "张三"， "年龄"： 20｝。

# 访问元素
印 学生["姓名"]。

# 修改元素
学生["年龄"] = 21。

# 添加元素
学生["成绩"] = 85。

# 遍历
遍历 键 于 学生.键()：
  印 键 "：" 学生[键]。
。
```

## 示例程序

### 1. 斐波那契数列

```
定 斐波那契 = 函 n：
  若 n小于2：
    返回 n。
  否则：
    返回 斐波那契(n减1)加斐波那契(n减2)。
  。
。

遍历 i 于 范围(10)：
  印 斐波那契(i)。
。
```

### 2. 冒泡排序

```
定 冒泡排序 = 函 列表：
  定 n = 列表长度(列表)。
  遍历 i 于 范围(n)：
    遍历 j 于 范围(0， n减i减1)：
      若 列表[j]大于列表[j加1]：
        定 临时 = 列表[j]。
        列表[j] = 列表[j加1]。
        列表[j加1] = 临时。
      。
    。
  。
  返回 列表。
。

定 测试列表 = 【64， 34， 25， 12， 22， 11， 90】。
印 冒泡排序(测试列表)。
```

### 3. 素数判断

```
定 是素数 = 函 n：
  若 n小于2：
    返回 假。
  。
  遍历 i 于 范围(2， n)：
    若 n取模i等于0：
      返回 假。
    。
  。
  返回 真。
。

遍历 num 于 范围(1， 21)：
  若 是素数(num)：
    印 num "是素数"。
  。
。
```
