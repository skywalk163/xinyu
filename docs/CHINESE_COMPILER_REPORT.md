```yan
    定义 字符 = 代码[位置]。
    
    # 跳过空白字符
    # 识别数字
    # 识别字符串
    # 识别关键字和标识符
    # 识别符号
    ...
  
  返回 tokens。
```

**辅助函数**：
- 是空白
- 是数字
- 是中文字符
- 是符号
- 获取码点

**识别函数**：
- 识别数字
- 识别字符串
- 识别关键字或标识符
- 识别符号

- ✅ 解析返回语句
- ✅ 解析表达式语句
- ✅ 解析二元操作
- ✅ 解析函数调用

**核心函数**：
```yan
函数 语法分析：
  参数 tokens。
  
  定义 状态 = {
    "tokens": tokens,
    "位置": 0,
    "当前行号": 1,
    "当前列号": 1,
  }。
  
  定义 语句列表 = []。
  
  当满足 未结束 状态：
    定义 语句 = 解析语句 状态。
    如果 语句 不等于 空 那么
      语句列表.添加 语句。
  
  返回 创建节点 "Program" {"语句": 语句列表} 1 1。
```

### 2. 完整的编译流程

**三阶段编译**：
1. 词法分析：代码 → Token序列
2. 语法分析：Token序列 → AST
3. 代码生成：AST → Python代码

**示例**：
```yan
定义 x = 5。
```

↓ 词法分析

```
[
  {类型: "VAR", 值: "定义", ...},
  {类型: "IDENTIFIER", 值: "x", ...},
  {类型: "ASSIGN", 值: "=", ...},
  {类型: "NUMBER", 值: "5", ...},
  {类型: "PERIOD", 值: "。", ...},
]
```

↓ 语法分析

```
{
  类型: "VarDef",
  数据: {
    名字: "x",
    初始值: {类型: "Number", 数据: {值: "5"}, ...},
  },
  ...
}
```

↓ 代码生成

```python
x = 5
```
