# 心语语言规范 (Language Specification)

## 版本
v2.0 - 双字关键字版本

## 1. 关键字

### 1.1 核心关键字

心语采用双字关键字，语义明确，易于理解：

| 关键字 | 英文对应 | 说明 |
|--------|---------|------|
| 定义 | var/let | 变量定义 |
| 函数 | function | 函数定义 |
| 如果 | if | 条件判断 |
| 真值 | true | 布尔真值 |
| 假值 | false | 布尔假值 |

### 1.2 语法标记

| 标记 | 英文对应 | 说明 |
|------|---------|------|
| 那么 | then | 条件分支 |
| 否则 | else | 否则分支 |
| 可选 | elif | 可选分支 |
| 循环 | for | 遍历循环 |
| 当满足 | while | 条件循环 |
| 重复 | repeat | 重复执行 |
| 继续 | continue | 继续循环 |
| 跳出 | break | 跳出循环 |
| 结束 | end | 结束标记 |
| 遍历 | in | 遍历元素 |
| 次数 | times | 重复次数 |
| 返回 | return | 返回值 |
| 参数 | param | 函数参数 |

### 1.3 兼容性

系统同时支持新旧两种语法：

**新语法（推荐）**：
```yan
定义 x = 5。
函数 平方：
  参数 x。
  返回 x 相乘 x。
```

**旧语法（兼容）**：
```yan
定 x = 5。
函 平方：
  接收 x。
  返回 x 乘 x。
```

## 4. 语法结构

### 4.1 变量定义

```yan
定义 变量名 = 值。
```

示例：
```yan
定义 x = 5。
定义 名字 = "张三"。
定义 列表 = [1, 2, 3]。
```

### 4.2 函数定义

```yan
函数 函数名：
  参数 参数1。
  参数 参数2。
  函数体。
  返回 结果。
```

示例：
```yan
函数 平方：
  参数 x。
  返回 x 相乘 x。

函数 求和：
  参数 a。
  参数 b。
  返回 a 相加 b。
```

### 4.3 条件语句

```yan
如果 条件 那么
  分支1。
否则
  分支2。
```

示例：
```yan
如果 x 大于 0 那么
  打印 "正数"。
否则
  打印 "非正数"。
```

### 4.4 循环语句

**遍历循环**：
```yan
循环 变量 遍历 列表：
  循环体。
```

示例：
```yan
循环 i 遍历 [1, 2, 3]：
  打印 i。
```

**条件循环**：
```yan
当满足 条件：
  循环体。
```

示例：
```yan
当满足 x 大于 0：
  打印 x。
  x = x 相减 1。
```

**重复执行**：
```yan
重复 次数 次数：
  循环体。
```

示例：
```yan
重复 5 次数：
  打印 "你好"。
```

## 5. 数据类型

### 5.1 基本类型

- **数字**：整数和浮点数
- **字符串**：文本数据
- **布尔值**：真值、假值
- **列表**：有序集合
- **字典**：键值对集合

### 5.2 类型推断

心语支持自动类型推断：

```yan
定义 x = 5。        # 推断为整数
定义 y = 3.14。     # 推断为浮点数
定义 s = "你好"。   # 推断为字符串
定义 b = 真值。     # 推断为布尔值
定义 l = [1, 2, 3]。 # 推断为列表
```

## 6. 注释

### 6.1 单行注释

使用 `#` 开头：

```yan
# 这是注释
定义 x = 5。  # 行尾注释
```

### 6.2 文档注释

使用三引号：

```yan
"""
这是文档注释
可以跨越多行
"""
函数 示例：
  参数 x。
  返回 x。
```

## 7. 错误处理

### 7.1 语法错误

当语法不符合规范时，会提示语法错误：

```
语法错误: Expected '那么' after condition (行 1, 列 6)
```

### 7.2 语义错误

当语义不正确时，会提示语义错误：

```
语义错误: Undefined variable 'x' (行 2, 列 5)
  💡 建议: Did you mean to define 'x' first?
```

## 8. 标准库

心语提供丰富的标准库：

### 8.1 数学库 (math)

```yan
导入 math。

math.平方根 16。    # 4.0
math.幂 2 3。       # 8.0
math.正弦 0。       # 0.0
```

### 8.2 字符串库 (string)

```yan
导入 string。

string.大写 "hello"。    # "HELLO"
string.小写 "WORLD"。    # "world"
string.分割 "a,b,c" "，"。 # ["a", "b", "c"]
```

### 8.3 列表库 (list)

```yan
导入 list。

list.排序 [3, 1, 2]。    # [1, 2, 3]
list.反转 [1, 2, 3]。    # [3, 2, 1]
list.求和 [1, 2, 3]。    # 6
```

## 9. 宏系统

心语支持强大的宏系统：

### 9.1 内置宏

- **遍历**：遍历列表元素
- **重复**：重复执行指定次数
- **持续**：持续执行（无限循环）
- **除非**：除非条件成立
- **当**：当条件成立时重复执行
- **断言**：断言条件为真
- **调试**：打印调试信息
- **计时**：测量执行时间

### 9.2 自定义宏

```yan
宏 双倍：
  参数 x。
  展开 x 相乘 2。

定义 y = 双倍 5。  # y = 10
```
