## 1. 词法结构

### 1.1 关键字

心语只有5个核心关键字：

| 关键字 | 含义 | 用途 |
|--------|------|------|
| `定` | 定义 | 定义变量或函数 |
| `函` | 函数 | 定义函数 |
| `若` | 如果 | 条件判断 |
| `真` | 真 | 布尔真值 |
| `假` | 假 | 布尔假值 |

### 1.2 语法标记

语法标记用于构建语法结构，不是关键字：

| 标记 | 用途 |
|------|------|
| `则` | 条件分支 |
| `否则` | 否则分支 |
| `否则若` | 否则如果 |
| `遍历` | 遍历循环 |
| `于` | 循环范围 |
| `当` | 当循环 |
| `重复` | 重复循环 |
| `次` | 次数 |
| `返回` | 返回语句 |

| `小于` | < | 小于 |
| `大于等于` | >= | 大于等于 |
| `小于等于` | <= | 小于等于 |

#### 逻辑操作符

| 心语 | Python | 说明 |
|------|--------|------|
| `且` | and | 逻辑与 |

### 1.4 分隔符

| 符号 | 名称 | 用途 |
|------|------|------|
| `，` | 逗号 | 分隔参数、列表元素 |
| `。` | 句号 | 语句结束 |
| `：` | 冒号 | 块开始 |
| `【】` | 方括号 | 列表 |
| `｛｝` | 花括号 | 字典 |
| `（）` | 圆括号 | 函数调用、分组 |

### 1.5 标识符

标识符用于命名变量、函数等：

- 可以包含中文、英文、数字、下划线
- 必须以中文或英文开头
- 区分大小写

```
有效标识符：
  姓名
  user_name
  变量1
  学生年龄

无效标识符：
  123变量    # 以数字开头
  @变量      # 包含特殊字符
```

| 类型 | 说明 | 示例 |

## 4. 语句

### 4.1 变量定义语句

```
定 变量名 = 值。
定 x = 10。
定 姓名 = "张三"。
```

### 4.2 赋值语句

```
变量名 = 值。
x = 20。
姓名 = "李四"。
```

### 4.3 函数定义语句

```
定 函数名 = 函 参数：
  函数体。
。

定 平方 = 函 x：
  返回 x乘x。
。
```

### 4.4 条件语句

```
若 条件：
  语句块。
。

若 条件：
  语句块1。
否则：
  语句块2。
。

若 条件1：
  语句块1。
否则若 条件2：
  语句块2。
否则：
  语句块3。
。
```

### 4.5 遍历循环语句

```
遍历 变量 于 可迭代对象：
  循环体。
。

遍历 i 于 【1， 2， 3】：
  印 i。
。
```

### 4.6 当循环语句

```

。

### 4.7 重复语句

```

  循环体。
。

  印 "你好"。

```

### 4.8 返回语句

```

```

### 4.9 表达式语句

```

## 5. 函数

### 5.1 函数定义

```
# 无参数函数
定 函数名 = 函：
  函数体。
。

# 单参数函数
定 函数名 = 函 参数：
  函数体。
。

# 多参数函数
定 函数名 = 函 参数1， 参数2， 参数3：
  函数体。
。
```

### 5.2 函数调用

```
函数名()。
函数名(参数)。
函数名(参数1， 参数2)。
```

### 5.3 递归函数

```
定 阶乘 = 函 n：
  若 n小于等于1：
    返回 1。
  否则：
    返回 n乘阶乘(n减1)。
  。
。
```

### 5.4 高阶函数

```
定 应用函数 = 函 f， x：
  返回 f(x)。
。

定 平方 = 函 x：
  返回 x乘x。
。

印 应用函数(平方， 5)。    # 输出：25
```

## 6. 作用域

### 6.1 全局作用域

在函数外部定义的变量具有全局作用域：

```
定 x = 10。              # 全局变量

定 函数 = 函：
  印 x。                 # 可以访问全局变量
。
```

### 6.2 局部作用域

在函数内部定义的变量具有局部作用域：

```
定 函数 = 函：
  定 y = 20。            # 局部变量
  印 y。
。

印 y。                   # 错误：y未定义
```

### 6.3 嵌套作用域

内部函数可以访问外部函数的变量：

```
定 外层 = 函：
  定 x = 10。
  
  定 内层 = 函：
    印 x。               # 可以访问外层的x
  。
  
  返回 内层。
。
```

## 10. 代码风格

### 10.1 命名规范

- 变量名：使用有意义的中文或英文
  ```
  定 姓名 = "张三"。
  定 user_name = "李四"。
  ```

- 函数名：使用动词或动词短语
  ```
  定 计算面积 = 函 ...
  定 get_data = 函 ...
  ```

### 10.2 缩进

使用2个空格缩进：

```
若 x大于0：
  印 "正数"。
。
```

### 10.3 空格

- 操作符两侧加空格：
  ```
  x加y        # 正确
  x加 y       # 错误
  ```

- 逗号后加空格：
  ```
  【1， 2， 3】    # 正确
  【1，2，3】     # 错误
  ```

### 10.4 语句结束

每个语句以句号 `。` 结束：

```
定 x = 10。
印 x。
```
