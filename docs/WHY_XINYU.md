### 1. 🀄 **纯中文编程**


### 2. 🎨 **元数驱动解析**

**世界首创的智能解析技术！**

```yan
# 自动识别参数数量
打印 "你好" "世界" "！"        # 3个参数
求和 1 2 3 4 5                # 5个参数
平方根 16                      # 1个参数

# 自动停止参数收集
斐波那契 n 相减 1 相加 斐波那契 n 相减 2
# 解析为：(斐波那契(n-1)) + (斐波那契(n-2))
```

**无需括号，无歧义，智能理解你的意图！**
