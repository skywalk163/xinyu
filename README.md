## 🌟 一眼惊艳

心语，寓意"从心而发，自然表达"。这是一门采用**元数驱动解析**技术的中文编程语言，让代码如心声般自然流淌。

```yan
# 传统语言
result = add(multiply(3, 5), sqrt(16))

# 心语语言
结果 = 3 相乘 5 相加 平方根 16
```

**没有括号的束缚，只有自然的表达！**

## 🎯 核心优势

### 1. 🀄 **纯中文编程**

用母语思考，用母语编程，思维零距离！

```yan
定义 价格 = 100。
定义 折扣 = 0.8。
定义 最终价格 = 价格 相乘 折扣。

如果 最终价格 小于 100 那么：
  打印 "太划算了！"。
否则：
  打印 "还是有点贵。"。
。
```

### 2. 🎨 **元数驱动解析** ⭐

**世界首创的智能解析技术！**

- 自动识别参数数量
- 无需括号，无歧义
- 智能理解你的意图

```yan
# 自动识别参数数量
打印 "你好" "世界" "！"        # 3个参数
求和 1 2 3 4 5                # 5个参数
平方根 16                      # 1个参数

# 自动停止参数收集
斐波那契 n 相减 1 相加 斐波那契 n 相减 2
# 智能解析为：(斐波那契(n-1)) + (斐波那契(n-2))
```

### 3. 🚀 **操作符动词**

中文操作符，自然流畅，像说话一样编程！

```yan
# 算术运算
3 相加 5        # 8
10 相减 3       # 7
4 相乘 5        # 20
20 相除 4       # 5

# 比较运算
5 等于 5        # 真值
10 大于 3       # 真值

# 逻辑运算
真值 并且 假值    # 假值
真值 或者 假值    # 真值
```

### 4. 📚 **丰富的内置函数**

28个内置动词，覆盖常用场景，开箱即用！

```yan
# 数学函数
平方根 16           # 4
绝对值 -5           # 5
最大值 1 5 3 2      # 5
求和 1 2 3 4 5      # 15

# 列表函数
定义 列表1 = 列表 1 2 3 4 5。
长度 列表1           # 5
范围 1 10            # [1,2,3,4,5,6,7,8,9]

# 类型转换
转整数 "42"          # 42
转字符串 123         # "123"
```

### 5. 🔧 **灵活的元数系统**

四种元数类型，适应所有场景！

| 元数类型 | 说明 | 示例 |
|---------|------|------|
| 固定元数 | 精确参数数量 | `平方根 16` |
| 可变元数 | 任意参数数量 | `打印 "a" "b" "c"` |
| 最小元数 | 最少参数数量 | `最大值 1 2 3` |
| 范围元数 | 参数范围 | `范围 1 10 2` |

### 6. 🎓 **易于学习**

零门槛，快速上手！5分钟学会基础，10分钟学会函数！

```yan
# 5分钟学会基础
定义 x = 10。              # 变量定义
打印 x。                    # 打印输出

# 10分钟学会函数
定义 平方 = 函 x：          # 函数定义
  返回 x 相乘 x。
。

打印 平方 5。               # 函数调用
```

创建 `hello.yan`：


## 📖 完整示例

### 基础示例

```yan
# 变量定义
定义 x = 10。
定义 y = 20。

# 算术运算
定义 和 = x 相加 y。
定义 差 = x 相减 y。
定义 积 = x 相乘 y。
定义 商 = x 相除 y。

# 打印结果
打印 和 差 积 商。
```

### 函数示例

```yan
# 定义函数
定义 平方 = 函 x：
  返回 x 相乘 x。
。

定义 立方 = 函 x：
  返回 x 相乘 x 相乘 x。
。

定义 两数之和 = 函 a b：
  返回 a 相加 b。
。

# 调用函数
打印 平方 5。          # 25
打印 立方 3。           # 27
打印 两数之和 10 20。   # 30
```

### 条件示例

```yan
定义 成绩 = 85。

如果 成绩 大于等于 90 那么：
  打印 "优秀"。
可选 成绩 大于等于 80 那么：
  打印 "良好"。
可选 成绩 大于等于 70 那么：
  打印 "中等"。
可选 成绩 大于等于 60 那么：
  打印 "及格"。
否则：
  打印 "不及格"。
。
```

### 循环示例

```yan
# 遍历循环
定义 数字列表 = 列表 1 2 3 4 5。
循环 数字 遍历 数字列表：
  打印 数字。
。

# 条件循环
定义 计数 = 0。
当满足 计数 小于 5：
  打印 计数。
  定义 计数 = 计数 相加 1。
。

# 重复执行
重复 5 次数：
  打印 "你好"。
。
```
