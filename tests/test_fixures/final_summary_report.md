## 🔍 发现的问题

### 高优先级问题（需深入修复）

1. **代码生成器bug**：斐波那契函数生成错误
   - **问题**：`斐波那契(((n - 1) + 斐波那契((n - 2))))`
   - **应该**：`斐波那契(n - 1) + 斐波那契(n - 2)`
   - **原因**：解析器处理函数调用和运算符优先级错误
   - **修复难度**：高（需要深入修改解析器）

2. **语法不支持**：多个边界测试失败
   - 函数参数语法不支持
   - 字典字面量语法不支持
   - while/for循环语法问题
   - **修复难度**：中

### 中优先级问题

3. **词法分析器问题**：
   - 不支持`\r`字符
   - 某些操作符不支持（如`<`）
   - 注释token类型不存在

4. **语义分析器问题**：
   - 函数外return语句未检测
   - 参数数量检查不完善
